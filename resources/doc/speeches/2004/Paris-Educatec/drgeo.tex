\section{Macro-construction}

\subsection{Introduction}
\begin{frame}
  \frametitle{Qu'est ce qu'une macro-construction~?}
  \begin{myDefinition}
    \begin{enumerate}
    \item Une macro-construction est une série de constructions
      enregistrée sous la forme d'une seule instruction. C'est la
      phase de \alert{construction} d'une macro.
    \item Lors de son utilisation, toute la série de constructions est
      réalisée en une fois. C'est la phase d'\alert{exécution} d'une
      macro.
    \item Cet enregistrement se dissocie de la figure pour être sauvé
      sur fichier.
    \end{enumerate}
  \end{myDefinition}

\pause

  \begin{myDefinition}
    C'est un outil pour apprendre à Dr.~Geo à faire des constructions complexes. 
  \end{myDefinition}
\end{frame}

\subsection{Conclusions}

\begin{frame}
  \frametitle{Que retenir ?}
  \begin{itemize}[<+-| alert@+>]
  \item  Créer une macro-construction, c'est apprendre à Dr.~Geo à faire
    une construction donnée.
  \item Une macro-construction attend des objets en entrée
  \item Elle retourne des objets en sortie, résultat d'une
    construction déduite à partir des objets en entrée
  \item Tous les objets intermédiaires de la construction sont masqués
    \item Les macro-constructions sont sauvées dans des fichiers,
      individuellement ou en groupe.
  \end{itemize}
\end{frame}

\section{Script}
\subsection{Introduction}

\begin{frame}
  \frametitle{Qu'est ce qu'un script~?}
  \begin{myDefinition}
    \begin{enumerate}[<+->]
    \item Un script est \alert{une fonction} écrite dans le langage
      Scheme. 
    \item Comme une fonction, il \alert{reçoit des arguments en entrée}.
    \item Toujours comme une fonction, il \alert{retourne une valeur
      numérique}.
    \item Cette valeur de retour est affichée dans la
      figure. Elle est utilisable par les constructions suivantes.
      \item Les arguments en entrée sont des références vers d'autres
        objets de la figue courante.
      \item Dr.~Geo propose une API -- interface de programmation --
      pour interroger ces objets.
    \end{enumerate}
  \end{myDefinition}
\end{frame}

\subsection{Conclusions}
\begin{frame}
  \frametitle{Que retenir~?}
  \begin{itemize}[<+-| alert@+>]
  \item  Un script est une fonction qui retourne une valeur affichée
    dans la figure.
  \item Il reçoit des arguments en entrée, références vers d'autres
    objets de la figure.
  \item Dr.~Geo propose une API pour interroger depuis un script des
    objets de la figure.
  \item Les scripts peuvent étendre les possibilités des
    macro-constructions~: les \emph{macro-scripts}.
  \end{itemize}
\end{frame}

\section{Figure programmée en Scheme}

\subsection{Introduction}

\begin{frame}
  \frametitle{Qu'est ce qu'une figure programmée~?}
  \begin{myDefinition}
    \begin{enumerate}[<+->]
    \item C'est une figure décrite dans un \alert{langage semi-naturel}. 
    \item Le langage informatique est \alert{Scheme}, une variante de
      \alert{Lisp}.
    \item Le langage permet une construction \alert{itérative} ou
      \alert{récursive} d'une figure.
    \item Dr.~Geo propose une \alert{API} -- interface de programmation --
      entièrement documentée.
    \end{enumerate}
  \end{myDefinition}
\end{frame}

\subsection{Exemples}

  \tiny
\begin{verbatim}
(nouvelle-figure "Spirale")

(define (square p1 p2 p3 p4 n)
   (let* ((s1  (Segment "" extremities     p1 p2))
          (s2  (Segment "" extremities     p2 p3))
          (s3  (Segment "" extremities     p3 p4))
          (s4  (Segment "" extremities     p4 p1))

          (B   (Point   "" sur-ligne    s2   1/10))
          (C   (Point   "" sur-ligne    s3   1/10))

     (envoi A masquer)
     (envoi B masquer)
     (envoi C masquer)
     (envoi D masquer)
 (if (> n 0)
     (square  A B C D (- n 1)))))

(soit Point "N" libre -5  5)

\begin{verbatim}
(nouvelle-figure "Triangle")

(define (triangle p1 p2 p3 n)
  (let* ((s1 (Segment "" extremities    p1 p2))
         (s2 (Segment "" extremities    p2 p3))
         (s3 (Segment "" extremities    p3 p1))
         (pe (Line    "" perpendiculaire  p3 s3))
         (ci (Circle  "" centre-segment p3 s2))
         (p4 (Point   "" intersection2  pe ci)))
    (envoi pe masquer)
    (envoi ci masquer)
    (envoi p4 masquer)
    (if (> n 0)
       (triangle p1 p3 p4 (- n 1)))))

(soit Point "O" libre  0  0)
(soit Point "A" libre -1  0)
(soit Point "B" libre -1  1)
(triangle O A B 15)
\end{verbatim}

(define (circle n)
(let*((r (Numeric "" libre 0 0 (/ 15 ( + 6 (* n n )))))
      (c (Point   "" libre (* 5 (/ 15 ( + 6 (* n n )))) 
                           (* 2 (* n (/ 15 ( + 6 (* n n )))))))
      (p (Circle "" centre-rayon c r )))
  (envoi r masquer)
  (if (> n 0)
    (circle (- n 1)))))

(circle 10)

(soit Point "O" libre 0   0)

(soit Line   "" 2points A O)
\end{verbatim}

   (envoi M masquer)          
   (if (> n 0)
      (begin
         (pythagore p4 M (- n 1))

     (envoi C masquer)
     (envoi D masquer)
 (if (> n 0)
     (square  A B C D (- n 1)))))

(soit Point "M" libre  5  5)
(soit Point "N" libre -5  5)
(soit Point "O" libre -5 -5)
(soit Point "P" libre  5 -5)

(square M N O P 2)
\end{verbatim}
\end{columns}
\end{frame}

\begin{frame}[fragile]
  \frametitle{Spirale -- Principales commandes 1/}

\begin{myCode}
\begin{verbatim}
 (soit Point "M" libre 5 5)
\end{verbatim}
\end{myCode}

D\'eclaration d'un point libre, nomm\'e M et de coordonn\'ees (5;5).
\alert{\texttt{soit}} est un mot-cl\'e pour cr\'eer un objet, l'objet cr\'e\'e
est accessible depuis la variable \texttt{M}.

\end{frame}

\begin{frame}[fragile]
  \frametitle{Spirale -- Principales commandes 2/}

\begin{myCode}
\begin{verbatim}
(let* ((s1  (Segment "" extremities p1 p2))
       (A   (Point   "" sur-ligne  s1   1/10))
\end{verbatim}
\end{myCode}

\texttt{let*} contient un bloc de d\'eclaration de variables locales
\`a la fonction \texttt{square}.

Au lieu du mot-cl\'e \texttt{soit}, des \alert{appels de fonctions} permettent
de cr\'eer les objets g\'eom\'etriques.

Le r\'esultat de ces appels est affect\'e aux variables \texttt{s1},
\texttt{A}, etc.

\end{frame}


\begin{frame}[fragile]
  \frametitle{Spirale -- Principales commandes 3/}

\begin{myCode}
\begin{verbatim}
(Point   ""  sur-ligne  s1  1/10)
\end{verbatim}
\end{myCode}

Cet appel cr\'ee un point sur la ligne \texttt{s1} avec comme abscisse
curviligne 1/10. 

Quelque soit le type de ligne (droite, demi-droite,
cercle, etc), les points qui y sont plac\'es ont une abscisse
curviligne comprise dans [0;1]. 

\end{frame}
